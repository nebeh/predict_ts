1. Симулятор рынка возвращает данные: день, цена, количество продаж. Описание симулятора смотри в comparison.ipynb

2. Делаем предсказание на предобученных (zero-shot предсказания)  на следующих моделях:
-  Amazon Chronos
-  Google TimesFM
-  PatchTST
-  iTransformer
-  Temporal Fusion Transformer

3. Предсказания на дообученных (в очень первом приближении) моделях (результаты на нескольких симуляциях см в папке images). 
-  Amazon Chronos
-  Temporal Fusion Transformer
-  LSTM с выделением трендов, сезонности и модели спроса
